    Mario 64 Rivals

    Mario 64 Rivals is an open-source web application written by Matt Wiens for competing against your best friends (dawww ) in Mario 64.

    How do I join?

    This instance of Mario 64 Rivals is meant to be closed community, in the sense that all users know each other in person. If you're such a person, contact me and I'll hook you up with an account. If you're not such a person, feel free to host your own instance of Mario 64 Rivals .

    Where's the source code?

    The source code can be found on Github here. It's licensed under the BSD 3-Clause license, which you can find here.
